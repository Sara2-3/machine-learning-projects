# Machine Learning Projects

A comprehensive collection of Machine Learning and Deep Learning projects, structured to guide you through the entire ML pipeline — from raw data to advanced neural network architectures.

---

## Overview

This repository contains a systematic approach to machine learning, organized into distinct modules that represent the complete workflow of an ML project. Each folder serves a specific purpose in the data science lifecycle, making it easy to navigate and understand the progression from data exploration to model deployment.

---

## Project Structure

```
├── data/
│   ├── raw/                  # Datasets and raw data
│   ├── analysis/             # Exploratory analysis and visualizations
│   └── preprocessing/        # Data cleaning and transformation
│
├── core-ml/
│   ├── supervised/
│   │   ├── regression/       # Linear, Polynomial, Ridge, Lasso, Random Forest
│   │   └── classification/   # Logistic, KNN, SVM, Decision Tree, Naive Bayes
│   └── unsupervised/         # K-Means, DBSCAN, PCA, t-SNE
│
└── deep-learning/
    ├── fundamentals/         # Activation functions, loss, optimizers, backprop
    ├── mlp/                  # Multi-Layer Perceptron
    ├── cnn/                  # Convolutional Neural Networks
    └── future/               # RNN, LSTM, Transformers, GANs
```

## Module Descriptions

### Data Science Analysis
- Exploratory Data Analysis (EDA)
- Statistical visualizations
- Pattern and correlation identification
- Distribution analysis

### Data Preprocessing
- Data cleaning (missing values, outliers)
- Feature engineering
- Normalization and standardization
- Categorical variable encoding
- Train-test splitting

- Random Forest Regression
